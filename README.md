# Sidequest 5 – Meditative Camera World

## Description

This project is a reflective camera experience that scrolls through a world larger than the screen. The camera follows the player with slow, smooth motion to create a calm, meditative feeling. As the camera moves, small heart symbols are subtly revealed throughout the world.

## Interaction Instructions

- Use **WASD** or **arrow keys** to move
- The camera scrolls smoothly as you explore
- Move slowly to notice hidden heart symbols as they appear

## Assets

- All visuals are generated using p5.js shapes
- No external image or sound assets were used
